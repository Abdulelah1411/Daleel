مشروع Daleel ⚽

نظرة عامة
Daleel هو نظام ذكي يعتمد على الذكاء الاصطناعي وتحليل البيانات لتحسين إدارة فرق كرة القدم، من خلال تقديم تحليلات عميقة لأداء اللاعبين، واقتراح التشكيلة المثالية، وتحديد الاستراتيجيات التكتيكية الأمثل لكل مباراة.

أهداف المشروع
- تحليل متقدم لأداء اللاعبين باستخدام الذكاء الاصطناعي.
- تقديم اقتراحات تكتيكية مبنية على بيانات الفريق المنافس.
- اختيار التشكيلة المثالية بناءً على أداء اللاعبين والظروف المحيطة بالمباراة (مكان المباراة، نوع المباراة، قوة المنافس).
- توقع أداء اللاعبين (الأهداف، البطاقات، التسديدات).

المميزات الأساسية
- لوحة تحكم (Dashboard): نظرة شاملة على أداء الفريق وتوصيات فورية.
- تحليل الأداء الفردي والجماعي: تقييمات دقيقة للاعبين والفرق.
- تشكيلة مثالية: اختيار أفضل 11 لاعبًا لكل مباراة حسب البيانات والتحليلات.
- اقتراحات تكتيكية: اقتراحات حول الخطة (هجومية، متوازنة، دفاعية) حسب ظروف المباراة.

الأدوات والتقنيات المستخدمة
- Python
- Pandas, NumPy, Scikit-learn
- Matplotlib, Seaborn (للعروض البصرية)
- Random Forest, XGBoost (نماذج تعلم الآلة)
